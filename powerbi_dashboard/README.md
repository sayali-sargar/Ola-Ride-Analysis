# Power BI Dashboard Overview

Power BI was used to visually present insights derived from the SQL analysis of Ola ride booking data.

## Dashboard Objective

The objective of the dashboard is to provide an easy-to-understand view of ride performance, cancellations, revenue patterns, and service quality metrics.

## Key Areas Covered

- Overall booking trends and booking status distribution
- Ride cancellations by customers and drivers
- Revenue contribution and booking value analysis
- Ride distance patterns across vehicle types
- Customer and driver ratings to assess service quality

## How the Dashboard Adds Value

The dashboard helps transform raw SQL outputs into visual insights, making it easier to identify trends, compare performance across categories, and communicate findings to non-technical stakeholders.
